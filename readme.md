# Imagura

Быстрый асинхронный просмотрщик изображений, построенный на Python и raylib.

## Описание

Imagura — это минималистичный, но мощный просмотрщик изображений с акцентом на производительность и удобство. Благодаря асинхронной загрузке с приоритизацией и GPU-ускорению через raylib, приложение обеспечивает плавную работу даже с большими галереями.

### Основные возможности

- **Асинхронная загрузка**: Приоритетная очередь загрузки (текущее изображение > соседние > остальные)
- **GPU-ускорение**: Быстрый рендеринг через raylib
- **Галерея с миниатюрами**: Сглаженные анимации при навигации
- **Навигация**: Зум и панорамирование с помощью мыши и клавиатуры
- **Режимы фона**: Прозрачный, темный, светлый
- **Поддержка EXIF**: Опциональная поддержка при установке Pillow
- **Оптимизация для Windows**: Корзина для удаления файлов, эффекты размытия
- **Интерфейс настроек**: Живой предпросмотр параметров
- **Горячие клавиши**: Удобное управление с клавиатуры

## Установка

### Требования

- Python 3.7+
- raylib привязка для Python

### Базовая установка

```bash
pip install pyray
```

### С поддержкой EXIF (опционально)

```bash
pip install pyray Pillow
```

## Использование

Запустите просмотрщик из командной строки:

```bash
python imagura2.py [путь]
```

**Примеры:**

```bash
python imagura2.py ~/Pictures           # Открыть папку
python imagura2.py ~/Pictures/photo.jpg # Открыть конкретный файл
python imagura2.py .                    # Открыть текущую папку
```

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| **←** / **→** | Навигация по изображениям |
| **Space** | Воспроизведение/пауза (GIF, видео) |
| **Z** | Вписать в окно |
| **1** / **2** / **3** | Актуальный размер / Масштаб к ширине / Масштаб к высоте |
| **R** | Повернуть по часовой |
| **E** | Повернуть против часовой |
| **D** | Удалить текущее изображение |
| **S** | Открыть параметры |
| **Esc** | Выход |
| **+** / **-** | Увеличить/уменьшить масштаб |
| **Колесо мыши** | Зум |
| **Drag** | Панорамирование |
